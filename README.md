# textbook-landing-page
first html project - mock product landing page
This project is the starting point of my journey toward becoming a Front-End Developer.

It is a mock product landing page built using pure HTML, created to practice semantic structure, content hierarchy, and basic user experience layout.


Purpose:

The goal of this project was to:

Understand how to structure a real-world product page

Practice writing clean, semantic HTML

Organize content in a clear and logical way

Build a mobile-friendly single-column layout


This marks my first step into professional web development.



Technologies Used:

HTML5

Semantic elements (header, section, footer, etc.)



 Layout & Structure:

The page follows a clean, single-column structure that adapts well across devices and avoids horizontal scrolling.
